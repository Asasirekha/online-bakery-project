# Bakery Online Ordering Website

## Project Overview
This project is a responsive online bakery ordering website built using HTML, CSS, and PHP.  
It allows users to browse bakery products and place orders through a simple, user-friendly form.

## Features
- Clean and intuitive user interface  
- Responsive design for both desktop and mobile devices  
- Order placement form handled with PHP backend  
- Displays bakery items with prices and images  

## Technologies Used
- HTML5 for website structure  
- CSS3 for styling and responsiveness  
- PHP for backend order processing  

## How to Run Locally
1. Download or clone the project files.  
2. Install a local server environment like XAMPP, WAMP, or MAMP.  
3. Copy the project folder into the `htdocs` (or equivalent) directory of your local server.  
4. Start the Apache server using the control panel of your server environment.  
5. Open your browser and go to `http://localhost/your-project-folder` to view and test the website.
